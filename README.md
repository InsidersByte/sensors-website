## Development

* `nvm use` || Install [Node.js](https://nodejs.org/en/)
* Install [Yarn](https://yarnpkg.com/) (npm will work too)

```bash
# Install the dependencies
yarn install

# Run the server
yarn start-server

# Run the website
yarn start

# Run the tests
yarn flow
yarn test
```
